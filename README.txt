///////////////////////////// COMMANDES

Z : Avancer
Q : Aller à gauche
S : Reculer
D : Aller à droite
A : Attaquer (pas encore d'animation)
E : Ouvrir l'inventaire

///////////////////////////// CONTENU

L'exécutable se trouve dans le dossier "Build" et se nomme "projectcell.exe", il suffit simplement de le lancer pour accéder au jeu.
Pour le moment, le jeu contient un étage généré aléatoirement à chaque démarrage et qui contient une salle de départ (grise), une salle au trésor (dorée) et une salle de boss (rouge) avec deux ennemis.
Il est aussi possible d'afficher les statistiques données par l'objet récupéré dans la salle au trésor en ouvrant l'inventaire. Le nom de l'objet ainsi que ses statistiques seront alors affichées.

Les scripts utilisés dans le jeu sont accessibles dans le dossier Assets/Scripts et sont rédigés en C# via VisualStudio.
